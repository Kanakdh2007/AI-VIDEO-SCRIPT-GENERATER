# AI-Powered Video Script Generator

## Project Overview
This project is an AI-powered web application that generates marketing video scripts from product details. Users enter information about a product, target audience, and campaign goal, and the system creates a complete video script with scene suggestions and a call-to-action.

## Features
- AI-generated marketing video scripts
- Multiple scene suggestions
- Customizable script duration
- User-friendly web interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Generative AI
- Prompt Engineering
